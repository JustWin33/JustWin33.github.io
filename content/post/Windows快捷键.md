# Windows快捷键完整功能列表

由于Windows快捷键数量庞大（不同版本、场景下超过千种），本列表系统整理**最核心、最常用的系统级快捷键**，按功能分类并标注适用版本。建议收藏作为速查手册。

---

## 一、Win键组合（Windows 10/11核心）

### 基础系统操作
| 快捷键                | 功能描述                           | 版本说明                |
| --------------------- | ---------------------------------- | ----------------------- |
| **Win**               | 打开/关闭开始菜单                  | 全版本                  |
| **Win + A**           | 打开操作中心/快速设置              | Win10/11                |
| **Win + I**           | 打开设置面板                       | Win10/11                |
| **Win + E**           | 打开文件资源管理器                 | 全版本                  |
| **Win + R**           | 打开"运行"对话框                   | 全版本                  |
| **Win + S / Win + Q** | 打开搜索/Cortana                   | Win10/11（S/Q功能相同） |
| **Win + L**           | 锁定计算机                         | 全版本                  |
| **Win + D**           | 显示桌面（再按恢复）               | 全版本                  |
| **Win + M**           | 最小化所有窗口                     | 全版本                  |
| **Win + Shift + M**   | 还原最小化的窗口                   | 全版本                  |
| **Win + ,**           | 临时查看桌面（松手恢复）           | Win7+                   |
| **Win + V**           | 打开剪贴板历史记录                 | Win10 1809+             |
| **Win + X**           | 打开高级用户菜单（含设备管理器等） | Win8+                   |
| **Win + .**           | 打开表情符号面板                   | Win10+                  |
| **Win + H**           | 打开听写/共享功能                  | Win10/11                |

### 窗口与任务管理
| 快捷键                 | 功能描述                       |
| ---------------------- | ------------------------------ |
| **Win + Tab**          | 打开任务视图（时间线）         |
| **Win + Ctrl + D**     | 创建新的虚拟桌面               |
| **Win + Ctrl + F4**    | 关闭当前虚拟桌面               |
| **Win + Ctrl + ←/→**   | 切换虚拟桌面                   |
| **Win + 1/2/3...**     | 打开任务栏第1/2/3个固定程序    |
| **Win + Shift + 数字** | 打开新实例（同Shift+中键点击） |
| **Win + Alt + 数字**   | 打开程序的跳转列表             |
| **Win + T**            | 循环切换任务栏程序（无预览）   |
| **Win + Shift + T**    | 反向切换任务栏程序             |

### 窗口布局（Win7+）
| 快捷键                | 功能描述                     |
| --------------------- | ---------------------------- |
| **Win + ←/→**         | 将窗口贴靠到屏幕左右半侧     |
| **Win + ↑**           | 最大化窗口                   |
| **Win + ↓**           | 最小化/还原窗口              |
| **Win + Shift + ←/→** | 将窗口移动到另一显示器       |
| **Win + Home**        | 最小化除当前窗口外的所有窗口 |

### 系统功能
| 快捷键                | 功能描述                            |

| **Win + G**           | 打开Xbox Game Bar                   |

## 版本差异说明

**Win+X菜单演变：**
- **Win7**：打开Windows移动中心
- **Win8/8.1**：打开高级用户菜单（含设备管理器、命令提示符等）
- **Win10/11**：功能更丰富，含PowerShell选项；**Win11**支持访问键（按字母快速选择）

**Win+Tab演变：**
- **Win7**：Flip 3D效果
- **Win10/11**：任务视图（虚拟桌面管理）

**Win+C演变：**
- **Win10**：激活Cortana语音
- **Win11**：打开Teams聊天

## 使用技巧

1. **Win+X + 字母**：在Win11中，按Win+X后按对应字母可快速打开功能（如D=设备管理器，T=任务管理器）
2. **Win+V**：可访问剪贴板历史，包含之前复制的文本和图片
3. **Ctrl+Shift+Esc**：比Ctrl+Alt+Delete更快打开任务管理器
4. **Win+.** ：不仅可输入表情，还有特殊符号和颜文字

---

**注意**：部分快捷键可能因键盘布局、系统版本或软件冲突而有所不同。第三方软件（如QQ、微信）可能占用全局快捷键。建议优先掌握**加粗**的常用组合，可大幅提升操作效率。
