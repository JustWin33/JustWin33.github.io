# Windows 运行命令（Win+R）完整功能列表

以下整理了Windows系统中最常用和实用的运行命令，按功能分类并标注使用说明。

---

## 一、系统核心命令

| 命令               | 功能描述           | 备注                     |
| ------------------ | ------------------ | ------------------------ |
| `msconfig`         | 系统配置实用程序   | 管理启动项、引导选项     |
| `regedit`          | 注册表编辑器       | **慎用**，修改前建议备份 |
| `gpedit.msc`       | 组策略编辑器       | **仅专业版/企业版**      |
| `taskmgr`          | 任务管理器         | 比Ctrl+Shift+Esc更直接   |
| `appwiz.cpl`       | 程序和功能         | 卸载软件                 |
| `control`          | 打开控制面板       | 新版Win10/11建议用设置   |
| `sysdm.cpl`        | 系统属性           | 环境变量、远程设置等     |
| `shutdown -s -t 0` | 立即关机           | `-r`参数为重启，`-a`取消 |
| `cmd`              | 命令提示符         | 经典命令行工具           |
| `powershell`       | Windows PowerShell | 更强大的命令行环境       |
| `wt`               | Windows终端        | **Win10/11**现代化终端   |

## 七、文件夹快捷方式

在运行框中可直接输入以下路径快速打开：

| 命令             | 功能描述                            |
| ---------------- | ----------------------------------- |
| `.`              | 打开当前用户主文件夹                |
| `..`             | 打开"此电脑"                        |
| `%temp%`         | 临时文件夹                          |
| `%appdata%`      | 应用数据文件夹（Roaming）           |
| `%localappdata%` | 本地应用数据文件夹                  |
| `%userprofile%`  | 当前用户文件夹                      |
| `%systemroot%`   | Windows系统目录（通常是C:\Windows） |

## 使用技巧

1. **管理员权限**：在运行框输入命令后，按 `Ctrl + Shift + Enter` 可以管理员身份运行
2. **历史记录**：运行框会保留历史输入，按 `Win + R` 后下拉查看
3. **快速复制**：按 `F4` 键可快速选中运行框中的地址栏内容
4. **命令参数**：多数命令支持参数（如`cmd /k`执行后不关闭窗口），可在命令后加`/?`查看帮助

---

**注意**：
- 部分命令（如`gpedit.msc`）仅在Windows专业版/企业版中可用
- 修改注册表或系统配置前务必备份数据
- 新版Windows正逐步将功能从控制面板迁移至设置应用
